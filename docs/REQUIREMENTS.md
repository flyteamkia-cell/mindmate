# MindMate Requirements

**MindMate** — An AI-powered task manager that intelligently prioritizes what the user should do next.

## User story: Sara the teacher

Sara teaches high school and juggles grading, lesson prep, parent emails, and after-school clubs. By Sunday night she has 15+ tasks scattered across notebooks and apps, and no idea where to start. She opens MindMate, and instead of a flat list, she immediately sees the 3 things she should tackle first — grading tomorrow's quiz, finishing Monday's lesson plan, replying to a parent — ranked by what actually matters, not just what was added last. She checks items off as she goes and searches for "field trip" to find a task she added weeks ago.

## MVP in scope

- User signup and login
- Create, edit, delete tasks
- Mark tasks done
- AI-powered prioritization
- Simple dashboard
- Search across tasks

## Out of scope for MVP

Mobile app, teams, email notifications, attachments, advanced calendar, voice reminders.

## Success criteria (MVP done)

- A user can sign up, log in, and log out.
- A user can create, edit, delete, and complete tasks.
- The dashboard shows an AI-ranked "do next" view, not just a raw list.
- Search returns matching tasks by keyword.
- The app is deployed and usable end-to-end by someone outside the project (e.g. Sara), not just running locally.
